Политика обработки персональных данных

1. Общие положения

Настоящая политика обработки персональных данных (далее — «Политика») определяет порядок и условия обработки данных, предпринимаемые платформой Vox-Meter (далее — «Оператор»), и устанавливает меры по обеспечению безопасности этих данных в рамках передовой цифровой инфраструктуры.

1.1. Соблюдение прав и свобод человека при обработке данных — важнейшее условие деятельности Vox-Meter. Мы обеспечиваем защиту прав на неприкосновенность частной жизни и личную тайну в соответствии со стандартами ЕС.

1.2. Настоящая Политика применяется ко всей информации, которую Оператор может получить о посетителях (далее — «Пользователи») веб-сайта vox-meter.blog.

2. Основные понятия

  • Веб-сайт — совокупность графических материалов, программ для ЭВМ и AI-алгоритмов, обеспечивающих доступность материалов по адресу vox-meter.blog.
  • Пользователь — любой посетитель, изучающий инновационные технологии на нашем ресурсе.
  • Персональные данные — любая информация, относящаяся к определенному Пользователю, включая данные для настройки AI-ассистентов.
  • Обработка данных — любые действия с данными (сбор, хранение, обезличивание), совершаемые с использованием систем автоматизации нового поколения.

3. Данные, которые мы обрабатываем

Оператор обрабатывает следующие категории данных для реализации ваших реальных задач:

  • Предоставляемые Пользователем:
    • Имя (для персонализации ответов AI);
    • Адрес электронной почты (для связи и доступа к платформе);
    • Номер телефона (только цифры, для подтверждения доступа во Франции).
  • Автоматический сбор:
    • Данные о поведении на сайте, собираемые через файлы «cookie» и технологии JavaScript для улучшения пользовательского опыта и аналитики.

4. Цели обработки

  • Идентификация Пользователя для предоставления доступа к закрытым разделам блога и кейсам.
  • Установление обратной связи для бесплатной консультации по вопросам разработки ботов.
  • Информирование о новых стратегиях роста и программах пассивного дохода на базе инноваций.
  • Улучшение качества работы AI-алгоритмов сайта на основе анонимных данных.

5. Правовые основания

Мы обрабатываем данные только на законных основаниях: при наличии вашего добровольного согласия (через формы на сайте) и в соответствии с регламентами безопасности ЕС, доступными в Европе.

6. Безопасность и хранение

  • Оператор принимает все технические меры для защиты данных от неправомерного доступа, используя экспертную поддержку на всех этапах.
  • Данные никогда не передаются третьим лицам без вашего явного согласия, за исключением случаев, предусмотренных законодательством Франции.
  • Срок обработки данных неограничен, но вы можете отозвать согласие в любой момент.

7. Заключительные положения

Пользователь может получить любые разъяснения по вопросам обработки данных, направив запрос Оператору на электронный адрес: hello@vox-meter.blog с пометкой «Запрос по данным».

Актуальная версия Политики расположена в свободном доступе на сайте vox-meter.blog. Технологии меняются, и мы будем отражать эти изменения в данном документе своевременно.
